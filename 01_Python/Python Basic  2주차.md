## 1. Jupyter Interpreter

### 1.1. install jupyter

> $ pip install jupyter



### 1.2. execute jupyter

> $ python -m notebook



### 1.3. shortcut

> `H` : shortcut list
>
> `Y` : change cell to code
>
> `M` : change cell to markdown
>
> `A` : insert cell above
>
> `B` : insert cell below
>
> `S` : save and Checkpoint
>
> `Z` : undo cell deletion
>
> `D, D` : delete selected cells
>
> `enter` : enter edit mode
>
> `ctrl + enter` : run selected cells
